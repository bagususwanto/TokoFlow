# Alur Pengguna

1. **Login**

   - Kasir atau Admin masuk dengan username/password.

2. **Penjualan**

   - Pilih barang → Masukkan jumlah → Simpan transaksi → Cetak struk.

3. **Manajemen Barang**

   - Admin dapat menambah atau memperbarui stok.

4. **Laporan**
   - Admin melihat laporan penjualan harian.
